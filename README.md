# flask-calculator-docker
Containerized Flask web application using Docker. This simple calculator supports addition, subtraction, multiplication, and division. The main focus is on containerizing a Flask web app using Docker with a custom Dockerfile and dependency management via requirements.txt.

**Tech Stack**
- Python
- Flask
- Docker

**What I Learned**
- Writing a Dockerfile
- Using requirements.txt for dependencies
- Container networking basics
- Running Flask inside Docker

**To run**

- docker build -t tanishrai01/calc .
- docker run -p 8000:5000 tanishrai01/calc
